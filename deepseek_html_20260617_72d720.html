API Documentation

Shorten URLs programmatically with our REST API. All endpoints require an API key.

Authentication

Pass your API key in the x-api-key header.

x-api-key: sk_sarthak_xxxxxxxxxxxx

Endpoints

POST /api/shorten – Create a short URL

{
                          "url": "https://example.com",
                          "custom": "myalias"   // optional
                        }

Response:

{
                          "short": "https://sarthakurll.vercel.app/myalias"
                        }

POST /api/generate-key – Generate a new API key

Response:

{
                          "apiKey": "sk_sarthak_xxxxxxxxxxxx"
                        }

Code Examples

JavaScript (fetch)

fetch("https://sarthakurll.vercel.app/api/shorten", {
                      method: "POST",
                      headers: {
                        "x-api-key": "YOUR_API_KEY",
                        "Content-Type": "application/json"
                      },
                      body: JSON.stringify({
                        url: "https://google.com"
                      })
                    })

cURL

curl -X POST https://sarthakurll.vercel.app/api/shorten \
                      -H "x-api-key: YOUR_API_KEY" \
                      -H "Content-Type: application/json" \
                      -d '{"url":"https://google.com"}'

Python

import requests

                    response = requests.post(
                        "https://sarthakurll.vercel.app/api/shorten",
                        headers={"x-api-key": "YOUR_API_KEY"},
                        json={"url": "https://google.com"}
                    )
                    print(response.json())
